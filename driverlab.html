Nautilus DriverLab

see all projects

Project Title:

DriverLab

Project Partners:

Dilan Nair

Project Sponsor:

Computer Science 343, Operating Systems

Goal:

Our goal was to create device drivers for a Graphics Processing Unit, and devices connected to the parallel port (such as a keyboard) for the Nautilus kernel. The final product was the ability to type letters and draw on the screen of a virtual machine, adding capabilities to the Nautilus operating system handout.

Status:

Complete; future tasks include drivers for other ports, and creating functionality in the operating system to chain input and output (e.g., typing in fonts).
